Resumo

Graduado como Tecnólogo em Análise e Desenvolvimento de Sistemas (2016) pela Universidade do Estado do Pará (UEPA). Atualmente estou me especializando em Desenvolvimento de Sistemas Web pela Estácio em Belém/PA. Trabalho com desenvolvimento front-end e back-end há 1 ano.

Minhas especialidades incluem o desenvolvimento de websites com códigos semânticos utilizando HTML5 e CSS3, construção de layouts responsivos fazendo uso de frameworks como o Bootstrap, desenvolvimento orientado a objetos utilizando a linguagem PHP e versionamento de código utilizando o Git. Possuo maior experiência na área de Tecnologia de Informação com ênfase em Tecnologias Web e Acessibilidade na Web.

Estou em constante aprendizado, afim de produzir cada vez mais produtos de qualidade.
